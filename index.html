cmines by Mortal

Minesweeper AI in n dimensions redux

Inspired by Igor Galochkin's Minefield 6D game, this is a Minesweeper-solving AI that handles the traditional 2 dimensional field as well as 3-, 4- and 5-dimensional fields (more dimensions are possible, but take a long time to play and solve).

Dependencies

Depends on ncurses for display. Although this can be disabled at runtime, it is required on compile time.

Install

make and run ./cmines.
